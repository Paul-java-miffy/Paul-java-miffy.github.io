hexo 框架小白搭建博客

第一步

  1. 到nodejs.org下载node

  2. cmd管理员启动 node -v 和 nmp -v查看版本

  3. 借助npm包管理器 安装 hexo 先输入

  4. 先安装cnmp 淘宝的镜像源 cnmp -v查看版本

  5. 安装hexo框架 hexo -v查看

    cnpm install -g hexo-cli

第二步

  1. hexo框架安装好后 新建一个文件夹 ，以后的操作都在这个文件夹里，以后出问题了删掉重来
  2. hexo init 创建博客
  3. hexo s 启动博客
  4. hexo new/n “第一篇文章” 新建文章 创建的是md格式的文件
  5. hexo clean 然后 写完文章以后hexo g 生成一下 再重新启动 就会出现在博客上

部署到远端 用github免费

  1. github仓库的命名 必须是你的昵称.github.io

  2. 安装一下插件 cnpm install –save hexo-deployer-git

  3. 需要设置 _config.yml 用text打开都行 在最后设置 type: git || repo: 你仓库的地址|| branch: master

  4. 部署到远端hexo d 输入githun 账号密码

  5. //需要git设置用户名 邮箱 初始化过需要用： 

    git config --global --replace-all user.email "输入你的邮箱"   
git config --global --replace-all user.name "输入你的用户名"

  6. 没有初始化 使用：

    git config --global user.name "输入你的用户名"   
git config --global user.email "输入你的邮箱"

    写在最后

  7. git需要先安装
